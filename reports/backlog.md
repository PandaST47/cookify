### #3. Хуки персистенции
- [x] **#3.1** Добавлен `useCooked(userId?)` — Map<recipeId, ratingOrNull> с per-user storage (`cookify:cooked:{userId}`). Методы: `add / remove / rate / unrate / has / ratingOf`. _(2026-05-02 — #1.3)_
- [x] **#3.2** Добавлен `useRatings()` — глобальный аггрегат «среднее + кол-во оценок» по recipeId. Хранит **только** пользовательские оценки в `cookify:ratings`, на чтение подмешивает seed (`recipe.rating × recipe.ratingCount`). Методы: `getRating / submit / replace / withdraw`. _(2026-05-02 — #1.3)_
- [x] **#3.3** Добавлен `useFilteredRecipes(...)` — pure-функция фильтрации (см. CLAUDE.md §5). Реализована полная иерархия: tab → search → sidebar (per-group AND/OR + cross-group AND) → exclusions (с правилом «явный ингредиент бьёт исключение») → chosen ingredients (OR) → sort. _(2026-05-02 — #1.4)_
